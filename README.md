## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/Abhisheksingh17cyber/abhireact-portfolio.git

# Install dependencies
cd abhireact-portfolio
npm install

# Start development server
npm start

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
